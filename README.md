# 🎮 Rock Paper Scissors Game

## 📌 Project Overview

This is a simple **Rock Paper Scissors game** developed as part of a class project.
The game allows a user to play against the computer with real-time results.

---

## 🚀 Features

* 🧑‍💻 User vs Computer gameplay
* 🎲 Random computer choice generation
* 📊 Score tracking system
* ⚡ Instant result display (Win / Lose / Draw)
* 🎨 Simple and interactive UI
